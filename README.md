# Travel Journal

## Overview

A simple and responsive React app that displays a list of travel destinations using modular components and dynamic data. Each destination includes an image, title, travel dates, description, and a link to view the location on Google Maps.

## Features

- Built with **React** and **Vite**
- Uses reusable components (`Header`, `Entry`)
- Dynamically renders content from a local `data.js` file
- Clean layout with organized CSS
- Image assets stored locally
